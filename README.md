# E-Commerce Retail Dashboard

## Project Overview

This project analyzes retail sales transactions to uncover business insights related to revenue generation, customer behavior, product performance, and sales trends. The analysis was performed using Python and visualized through an interactive Plotly dashboard.

The goal of this project is to demonstrate data cleaning, exploratory data analysis (EDA), KPI generation, business analytics, and dashboard development skills.

---

## Dataset

The dataset contains retail transaction records with the following attributes:

* Transaction ID
* Date
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price per Unit
* Total Amount

---

## Objectives

The project aims to answer the following business questions:

* What is the total revenue generated?
* How many transactions were completed?
* How many unique customers made purchases?
* Which product category generates the highest revenue?
* How do sales change over time?
* How does customer spending vary by gender and age?

---

## Key Performance Indicators (KPIs)

The following KPIs were calculated:

| KPI                       | Description                                 |
| ------------------------- | ------------------------------------------- |
| Total Revenue             | Total sales generated from all transactions |
| Total Orders              | Number of completed transactions            |
| Total Customers           | Number of unique customers                  |
| Total Quantity Sold       | Total units sold                            |
| Average Order Value (AOV) | Average revenue per transaction             |

## Exploratory Data Analysis

### Product Analysis

* Analyzed revenue generated by each product category.
* Identified the highest-performing product category.
* Compared category-wise sales contributions.

### Time Series Analysis

* Converted transaction dates into monthly periods.
* Analyzed monthly sales trends.
* Identified peak and low-performing sales periods.

### Customer Analysis

* Compared spending patterns across genders.
* Investigated the relationship between age and spending behavior.
* Identified major customer segments.
